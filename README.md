# Gemini AI REST API using Node.js & Express

An AI-powered REST API built using Node.js, Express.js, and Google's Gemini API.  
This project generates AI-based responses from user prompts and can be tested using Postman.

---

## Features

- REST API built with Express.js
- Gemini AI integration
- Prompt-based AI responses
- JSON API support
- Postman tested
- Environment variable security using dotenv

---

## Tech Stack

- Node.js
- Express.js
- Gemini API
- dotenv
- Postman
